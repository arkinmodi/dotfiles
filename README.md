# dotfiles

## Installation

1. Clone this repo

```sh
git clone --recurse-submodules https://github.com/arkinmodi/dotfiles.git ~/.dotfiles
```

2. Install `GNU stow` and `neovim`

```sh
brew install stow neovim
```

3. Setup symlinks

```sh
sh install.sh
```
